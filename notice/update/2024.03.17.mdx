<Note>

`Feature` Mile Visitors 사전 방문자 등록시, 개별 방문 활성화 기능이 추가되었습니다.  
`Fix` Mile Visitors 예약 방문 과정의 프로세스가 변경되었습니다. (인원 선택 방식)

</Note>

### Mile Visitors

---

    **웹**
    - 캘린더 일정 정보 등록 기능이 추가되었습니다.
    - 사전 방문자 등록 및 수정시 개인 캘린더 연동 기능이 추가되었습니다.
    - 방문 요청 승인시 모달로 개인 캘린더 연동 기능을 추가하였습니다.
    - 사전 방문자 등록 (임직원용) 방문 목적 select 가능하도록 기능을 추가하였습니다.
    - 방문 정보 등록 요청 링크로 이름을 수정하였습니다.

<br />

### Admin Web

---

    **공통**
    - 내 정보 관리 페이지에 개인 캘린더 연동 기능이 추가되었습니다.
    - 멤버 목록 동기화 버튼이 서비스 연동 페이지에서 멤버 권한 관리 페이지로 변경되었습니다.
    - 프로필 UI가 수정되었습니다.

    **Mile Visitors**
    - 사전 방문 목적 설정 기능이 추가되었습니다.
