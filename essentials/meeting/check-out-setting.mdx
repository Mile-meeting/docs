## 1. 정책 관리 페이지 접속하기

좌측 탭의 Meeting 아이콘을 클릭하신 후, 목록에서 **정책 관리**를 선택하여 페이지에 접속해주세요.

현재 설정할 수 있는 정책은 2가지로, 체크인과 체크아웃에 대한 정책을 설정하실 수 있습니다.

해당 페이지에서는 자동 체크아웃 정책 설정 관련해서 다루겠습니다.

## 2. 자동 체크아웃 기능 활성화

자동 체크아웃 기능 활성화 여부에 따라 태블릿에서 작동하는 방식은 아래와 같습니다.

-   **자동 체크아웃 기능 활성화 (BETA)**
    회의가 진행 중일때 (On-Air 화면) 회의실 내부에서 몇 분동안 소리가 감지되지 않으면 자동으로 체크아웃 처리됩니다. BETA 기능으로 오차가 있을 수 있으며, 회의가 진행 중인데 자동 체크아웃 알림이 뜬 경우 **화면을 터치**하여 회의를 계속 진행할 수 있습니다.

    <Info>
        ⚡ 자동 체크아웃 기능 활성화시, 마이크 권한이 없는 기기에 경우에 **녹음 권한을 허용하시겠습니까?** 팝업을 허용한 후 사용하실 수
        있습니다.
    </Info>

-   **자동 체크아웃 기능 비활성화**
    음성을 인식하지 않으며, 예약된 회의 종료 시간에 회의가 종료됩니다.

<Info>
    ℹ️ 자동으로 체크아웃되기 1분 전에, 정말 회의가 진행 중인 상태가 아닌지 확인하는 체크아웃 알림창이 띄워집니다. 회의가 진행 중인데
    알림창이 띄워진 경우 **화면을 터치하여 회의를 계속 진행**할 수 있습니다.
</Info>

## 3. 음성 감지 시간 설정

음성 감지 시간은 몇 분 이상 음성 감지가 되지 않았을 때 회의를 종료시킬 것인지를 판단하는 시간입니다.

<Info>⚡ 음성 감지 시간을 5분으로 설정한 경우, 회의실 내부에서 5분 이상 음성이 감지되지 않으면 자동 체크아웃 처리됩니다.</Info>

## 4. 자동 체크아웃 전 소리 알림 활성화

알림을 활성화하시면, 실제로 체크아웃되기 1분 전에 태블릿에서 알림창과 함께 알림 소리가 울립니다.

<Info>
    ⚡ 태블릿의 소리를 음소거로 해놓으셨을 경우, 해당 기능을 활성화해도 작동하지 않을 수 있으니, 해당 기능을 사용하시기 전에 꼭 **태블릿
    소리 크기를 확인해주세요! (미디어 소리)**
</Info>
