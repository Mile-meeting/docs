사전 방문자 등록시 개인 캘린더에 반영하는 방법에 대해서 자세히 보시려면, [개인 구글 캘린더에 사전 방문 정보 등록하기]()를 클릭해주세요!

## 1. 내 정보 관리 페이지 접속하기

좌측 탭에서 `Setting` 을 클릭해주시고, 목록 탭에서 `내 정보 관리` 를 클릭하여 해당 페이지로 접속해주세요.

    <img src="/images/essentials/link-private-calendar/img1.png" />
</Frame>

<br />

## 2. 개인 캘린더 연동하기

서비스 연동 섹션의 우측 `연동` 버튼을 클릭하여 연동해주세요.

버튼 옆에 **연동완료**로 표시되면, 정상적으로 연동이 완료됨을 뜻합니다.
