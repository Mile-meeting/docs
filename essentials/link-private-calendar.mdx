사전 방문자 등록시 개인 캘린더에 반영하는 방법에 대해서 자세히 보시려면, <a href="https://mile.mintlify.app/visitors/visitor-register/register-with-employee">개인 구글 캘린더에 사전 방문 정보 등록하기</a>를 클릭해주세요!

## 1. 내 정보 관리 페이지 접속하기

좌측 탭에서 `Setting` 을 클릭해주시고, 목록 탭에서 `내 정보 관리` 를 클릭하여 해당 페이지로 접속해주세요.

<Frame>
    <img src="/images/essentials/link-private-calendar/img1.png" />
</Frame>

<br />

## 2. 개인 캘린더 연동하기

서비스 연동 섹션의 우측 `연동` 버튼을 클릭하여 연동해주세요.

<Frame>
    <img src="/images/essentials/link-private-calendar/img2.png" />
</Frame>

<br />

버튼을 클릭하시면, 연결할 수 있는 개인 캘린더가 모달형식으로 나타나고, 연결하고 싶은 캘린더를 체크한 후, 연결 버튼을 클릭해주세요.

<Frame>
    <img src="/images/essentials/link-private-calendar/img3.png" />
</Frame>

<br />

연결이 완료되면 모달창이 꺼지며, 버튼이 `해제` 로 바뀝니다.

마찬가지로 개인 캘린더 연결을 해제하시고 싶으시면, **해제** 버튼을 클릭해주세요.

<Frame>
    <img src="/images/essentials/link-private-calendar/img4.png" />
</Frame>
