<Note>슬랙 연동은 슬랙 워크스페이스가 변동되거나, 추가 멤버가 들어왔을 시에 동기화가 필요합니다.</Note>

## 1. 메신저 연동 탭 접속하기

로그인 후 좌측의 설정 아이콘을 선택하신 뒤, `서비스 연동 관리 > 메신저 연동` 탭을 선택하여 메신저 연동 페이지로 접속합니다.

> 메신저 연동 탭은 슬랙과 구글챗과 같이, 마일 서비스 이용 시 메신저 전송이 필요한 기능에 사용되는 메신저를 연동하는 페이지입니다.

<Frame>
    <img src="/images/essentials/link/slack/img1.png" />
</Frame>

<br />

## 2. 슬랙 앱 워크스페이스에 연동하기

최초 연동 시에는 `연동` 버튼을 눌러 슬랙 연동 페이지로 이동합니다.

<Note>
    <img src="/images/essentials/link/slack/img2.png" />
</Note>

<br />
연동하시고자하는 슬랙의 워크스페이스를 우측 상단에서 설정해주시고, 하단 부에 있는 `허용` 버튼을 클릭해주시면 슬랙 연동이 완료됩니다.

<Frame>
    <img src="/images/essentials/link/slack/img3.png" />
</Frame>

<br />
해당 과정을 모두 완료하시면, 연동한 슬랙 워크스페이스에서 Mile 앱이 뜨는 것을 확인하실 수 있습니다.

<Frame>
    <img src="/images/essentials/link/slack/img4.png" />
</Frame>

<br />

연동 이후 임직원 슬랙 연동 상태만 갱신하고 싶으신 경우에는 `동기화` 버튼을 눌러 갱신할 수 있습니다.

<Frame>
    <img src="/images/essentials/link/slack/img5.png" />
</Frame>

<br />

## 3. 슬랙 연동 해제하기

슬랙 연동이 완료되면 아래처럼 `연동 해제` 버튼이 보여집니다. 메신저 연동 해제를 원하시는 경우 `연동 해제` 버튼을 선택합니다.

<Frame>
    <img src="/images/essentials/link/slack/img6.png" />
</Frame>

<br />
확인창이 뜨면 `연동 해제 버튼`을 눌러 연동 해제를 진행합니다.

<Warning>

메신저 연동을 해제하면 기존에 등록해놓은 메신저 알림 채널, 알림 데이터가 모두 삭제됩니다.

`Mile Meeting`

-   민원 알림 채널, 민원 담당자 데이터 삭제

`Mile Visitors`

-   초인종 기능 OFF / 초인종 알림 채널, 초인종 담당자 데이터 삭제
-   당일 방문 기능 OFF / 당일 방문 알림 채널, 당일 방문 담당자 데이터 삭제

</Warning>
