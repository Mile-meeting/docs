<Note>구글챗 메신저는 관리자가 연동한 이후에 임직원 개개인이 구글챗에서 앱 설치를 진행해야 원활한 이용이 가능합니다. 관련 가이드는 구글챗 연동하기에서 확인해주세요.</Note>

## 1. 메신저 연동 탭 접속하기

로그인 후 좌측의 설정 아이콘을 선택하신 뒤, `서비스 연동 관리 > 메신저 연동` 탭을 선택하여 메신저 연동 페이지로 접속합니다.

> 메신저 연동 탭은 슬랙과 구글챗과 같이, 마일 서비스 이용 시 메신저 전송이 필요한 기능에 사용되는 메신저를 연동하는 페이지입니다.

<Frame>
  <img src="/images/essentials/link/google-chat/img1.png" />
</Frame>

<br />

## 2. 구글챗 연동하기

구글챗 카드의 `연동` 버튼을 선택합니다.

<Frame>
  <img src="/images/essentials/link/google-chat/img2.png" />
</Frame>

<br />
확인창이 뜨면 `연동` 버튼을 눌러 연동을 완료합니다.

<Frame>
  <img src="/images/essentials/link/google-chat/img3.png" />
</Frame>

<br />

## 3. 구글챗 연동 해제하기

구글챗 연동이 완료되면 아래처럼 `연동 해제` 버튼이 보여집니다. 메신저 연동 해제를 원하시는 경우 `연동 해제` 버튼을 선택합니다.

<Frame>
  <img src="/images/essentials/link/google-chat/img4.png" />
</Frame>

<br />
확인창이 뜨면 `연동 해제` 버튼을 눌러 연동 해제를 진행합니다.

<Warning>
  메신저 연동을 해제하면 기존에 등록해놓은 메신저 알림 채널, 알림 데이터가 모두 삭제됩니다.</br>
  `Mile Meeting` </br>
  - 민원 알림 채널, 민원 담당자 데이터 삭제</br></br>

  `Mile Visitors` </br>
  - 초인종 기능 OFF / 초인종 알림 채널, 초인종 담당자 데이터 삭제 </br>
  - 당일 방문 기능 OFF / 당일 방문 알림 채널, 당일 방문 담당자 데이터 삭제 </br>
</Warning>
