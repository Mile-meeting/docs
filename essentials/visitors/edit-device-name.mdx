## 1. 기기 관리 페이지 접속하기

좌측의 `Mile Visitors` 탭을 클릭하고, 목록에서 `기기 관리` 페이지를 클릭하여, 해당 페이지에 접속해주세요.

기기 관리 페이지에서는 Visitors 제품에 등록된 기기 정보를 조회하고 각 기기별로 태블릿에서 활성화시킬 기능을 설정할 수 있습니다.

</Frame>

## 2. 설정할 기기 선택하기

설정하고자 하는 기기를 선택해주세요.

    <img src="/images/essentials/visitors/edit-device-name/img2.png" width="100%" />

## 3. 기기명 설정하기

첫 번째 탭인 기기 정보 탭에서 기기명을 수정할 수 있습니다. 기기명 우측에 위치한 수정 아이콘을 선택하여 수정을 진행해주세요.

<Info>

    사용하시는 경우, 해당 명칭을 설정하여 보다 더 효율적으로 솔루션을 사용해보세요!
</Info>

<Frame>
    <img src="/images/essentials/visitors/edit-device-name/img4.png" width="100%" />
</Frame>

<br />

수정하고자 하는 기기명을 작성한 뒤, 우측의 저장 버튼을 클릭해주시면, 최종적으로 기기명이 설정됩니다.

<Frame>
    <img src="/images/essentials/visitors/edit-device-name/img5.png" width="100%" />
</Frame>

<br />

## 4. 기기별 사용 기능 설정하기

기기 정보 탭의 `기능 설정` 영역에서 해당 기기의 기능 설정 현황을 조회할 수 있습니다.

상단에 위치한 탭을 선택하여 이동하거나 `기능 설정` 영역에서 원하는 기능을 선택하면 수정할 수 있는 탭으로 이동됩니다.

<Frame>
    <img src="/images/essentials/visitors/edit-device-name/img6.png" width="100%" />
</Frame>
