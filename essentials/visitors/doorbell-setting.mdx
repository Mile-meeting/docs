## 1. 기기 관리 페이지 접속하기

좌측의 `Mile Visitors` 탭을 클릭하고, 목록에서 `기기 관리` 페이지를 클릭하여, 해당 페이지에 접속해주세요.

기기 관리 페이지에서는 Visitors 제품에 등록된 기기 정보를 조회하고 각 기기별로 태블릿에서 활성화시킬 기능을 설정할 수 있습니다.

    <img src="/images/essentials/visitors/doorbell-setting/img1.png" width="100%" />
</Frame>

<br />

## 2. 설정할 기기 선택하기

설정하고자 하는 기기를 선택해주세요.

<Frame>
    <img src="/images/essentials/visitors/doorbell-setting/img2.png" width="100%" />
</Frame>

<br />
사이드 창이 띄워지면 초인종 탭을 선택해주세요.

<Frame>
    <img src="/images/essentials/visitors/doorbell-setting/img3.png" width="100%" />
</Frame>

<br />

## 3. 초인종 기능 활성화 여부 설정하기

우측에 위치한 토글 아이콘을 눌러 초인종 활성화 여부를 설정할 수 있습니다.

<Info>
    <aside>
        ℹ️ 이때 초인종 사용 여부에 따라서 기기의 메인 화면이 변경됩니다.
        <br />웹 상에서 메인 화면을 확인해보시려면, 홈 화면 구성 탭에서 현재 기기의 메인 화면을 보실 수 있습니다.
    </aside>
</Info>

<Frame>
    <img src="/images/essentials/visitors/doorbell-setting/img4.png" width="100%" />
</Frame>

<br />
## 4. 초인종 슬랙 채널 추가하기

초인종 슬랙 채널 섹션의 우측 `채널 추가하기` 버튼을 클릭합니다.

<Frame>
    <img src="/images/essentials/visitors/doorbell-setting/img5.png" width="100%" />
</Frame>

<br />

추가하고자 하는 채널의 링크를 가져와 붙여넣은 후 `조회` 버튼을 눌러주세요.  
조회된 슬랙 채널을 선택하고 `추가하기` 버튼을 누르면, 최종적으로 초인종이 눌렸을 때 알람 갈 슬랙 채널들이 추가됩니다.

상세한 슬랙 채널 추가 방법은 <a href="https://mile.mintlify.app/essentials/common/link-slack-channel">슬랙 채널 연결하기</a>에서 확인해주세요.

<Frame>
    <img src="/images/essentials/visitors/doorbell-setting/img6.png" width="100%" />
</Frame>

<br />

## 5. 초인종 담당자 추가하기

초인종 담당자 섹션의 우측 `담당자 추가하기` 버튼을 클릭합니다.

<Frame>
    <img src="/images/essentials/visitors/doorbell-setting/img7.png" width="100%" />
</Frame>

<br />
추가하고자 하는 담당자를 검색하고, 선택하여 추가하기 버튼을 클릭하면, 최종적으로 알림 받을 담당자가 추가됩니다.

<Frame>
    <img src="/images/essentials/visitors/doorbell-setting/img8.png" width="100%" />
</Frame>
