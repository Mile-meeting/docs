### 1. 초인종 기능 활성화 여부 변경하기

`Mile Visitors` 탭에서 `초인종 관리` 페이지에 접속하여 초인종 기능 여부를 설정할 수 있습니다.

<Frame>
    <img src="/images/visitors/doorbell-setting/img1.png" width="100%" />
</Frame>

<br />

우측 버튼을 통해 초인종 기능 활성화를 킵니다.

<Frame>
    <img src="/images/visitors/doorbell-setting/img2.png" width="100%" />
</Frame>

<br />

변경 확인 창이 뜨고, 최종적으로 `변경하기` 버튼을 클릭하면, 활성화 여부가 변경됩니다.

이때, 변경 확인 창에서 나오는 문구와 같이 초인종 기능 여부에 따라서 Mile Visitors 서비스의 메인 화면이 변경됩니다.

웹 상에서 메인 화면을 확인해보시려면, `테마 관리` 페이지에서 현재 Mile Visitors의 메인 화면을 보실 수 있습니다.

<Frame>
    <img src="/images/visitors/doorbell-setting/img3.png" width="100%" />
</Frame>

<br />
<br />

### 2. 초인종 슬랙 채널 추가하기

초인종 슬랙 채널 섹션의 우측 `채널 추가하기` 버튼을 클릭합니다.

<Frame>
    <img src="/images/visitors/doorbell-setting/img4.png" width="100%" />
</Frame>

<br />
추가하고자 하는 채널들을 선택하고 추가하기 버튼을 클릭하면, 최종적으로 방문자가 호출시 알람이 갈 슬랙 채널들이 추가됩니다.

<Frame>
    <img src="/images/visitors/doorbell-setting/img5.png" width="100%" />
</Frame>

<br />
<br />

초인종 담당자 섹션의 우측 `담당자 추가하기` 버튼을 클릭합니다.
