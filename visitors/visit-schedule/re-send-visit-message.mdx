---

## 1. 나의 방문 관리 페이지 접속하기

마일 사이트에서 로그인을 한 뒤, 좌측 탭 에서 `Mile Visitors` 선택후, 목록에 `나의 방문 관리` 페이지를 선택합니다.

`나의 방문 관리` 페이지는 관리자 뿐만이 아니라, 임직원 모든 분들이 접속이 가능한 페이지이며, Mile Visitors를 사용하시면서, 방문 예약/관리/조회 등등이 가능한 페이지입니다.

## 2. 재전송이 필요한 방문 상세보기

이전에 상세 안내 메시지 전송을 실패한 방문 건에 대해서 **더보기** 버튼을 클릭하여 방문 상세 모달을 띄워주세요.

<Frame>
    <img src="/images/visitors/visit-schedule/re-send-visit-message/img2.png" width="100%" />
</Frame>

<br />

방문 상세 모달에서는 **기본 방문 정보와 방문자 정보**를 확인하실 수 있습니다.

방문자 정보는 아래와 같은 정보를 제공합니다.

-   이름
-   연락처
-   방문자 상태
-   상세 안내 메시지 전송 채널 (카카오톡, 이메일)
-   상세 안내 메시지 전송 여부 (실패, 성공)

이때, 전송 여부가 `전송 실패` 의 경우 오른쪽에 `재전송` 버튼이 활성화되며, 해당 버튼을 클릭하여 재전송하실 수 있습니다.

<br />

<Frame>
    <img src="/images/visitors/visit-schedule/re-send-visit-message/img3.png" width="100%" />
</Frame>

## 3. 상세 안내 메시지 재전송하기

**재전송** 버튼을 클릭하시면, 아래와 같은 재전송 모달이 띄워집니다.

해당 모달에서, `안내 채널` 및 `연락처` 를 수정하고 `전송` 버튼을 클릭하시면 해당 입력하신 정보로 안내 메시지가 전송됩니다.
