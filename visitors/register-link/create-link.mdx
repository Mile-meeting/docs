---

방문객의 정보를 사전에 등록하는 방법은 크게 2가지가 있습니다.

-   `1. 사전 방문자 등록 방식` 기업의 담당자가 직접 방문객 정보를 입력하는 방식
-   `2. 방문 링크 전달 방식` 방문객에게 직접 방문 정보 입력을 요청하는 방식

해당 페이지에서는 `2. 방문 링크 전달 방식`에 대해서 알아보겠습니다.

사전 방문자 등록 방식에 대해서 자세히 알아보시려면, [사전 방문자 등록 방식](https://mile.mintlify.app/visitors/register-link/register)을 클릭해주세요.

<br />
<Frame>
    <img src="/images/visitors/register-link/create-link/img1.png" width="100%" />
</Frame>

우측 상단에 있는 자신의 **방문자 등록 링크**를 클릭하시고, 방문자에게 미리 링크를 공유해주세요.

기업 측에서 정보를 입력하실 필요 없이,
방문객에게 간편한 링크 공유만으로 방문 정보를 사전 등록할 수 있습니다.

<br />

<Frame>
    <img src="/images/visitors/register-link/create-link/img2.png" width="100%" />
</Frame>

방문객은 해당 정보를 사전에 입력하고 **등록 시점 직후와 방문 당일 오전 08:00에 카카오톡 알림과 리마인더**를 제공받을 수 있습니다.
