## 와야 할 내용

-   뒤 타임에 회의가 잡혀있는 상황 부터 연장요청 보내는 패스가 보여져야함

    -   여기서 최대 15분까지만 요청을 보낼 수 있다는 것을 알려줌

-   연장 요청을 보냈을 때 슬랙메시지가 전송되는 것을 보여줌
    -   여기서 슬랙 연장 요청 수락시에 뜨는 화면 보여줌
    -   기존 회의시간 종료 1분전에는 자동 거절로 판단한 다는 것을 보여줌
-   구글 캘린더에도 해당 내용이 반영된다는 것을 알려줘야 함
