title: "회의 연장 요청 보내기"
description: "바로 뒷타임 회의가 존재하는 경우, 연장 요청 기능을 통해 회의 이용 시간을 연장할 수 있습니다."
---

---

<Warning>
    1. 관리자 페이지의 `서비스 연동` 탭에서 먼저 슬랙을 연동해야 합니다.<br />
    2. 회의가 종료되기 5분 전까지만 연장 요청 기능을 사용하실 수 있습니다.<br />
    3. 연장 요청은 한 회의당 1회만 가능합니다.<br />
    4. 뒷타임 회의의 예약 시간이 15분 이하인 경우 연장 요청이 불가합니다<br />
</Warning>

## 1. 회의 연장 요청 보내기

타이머 안에 위치한 `연장하기` 버튼을 선택합니다.
<Frame>
    <img src="/images/tablet/meeting-extend/request-extend/img1.png" width="100%"/>
</Frame>

<br />

연장 시간이 뒷타임 회의 예약 시간과 겹치는 경우, `연장 요청` 버튼을 선택합니다.
<Frame>
    <img src="/images/tablet/meeting-extend/request-extend/img2.png" width="100%"/>
</Frame>
<br />
<Warning>
    연장 요청은 뒷타임 회의 기준으로 최대 15분만 가능하며, 연장 요청을 받은 상대방이 수락한 경우에만 회의 예약 시간이 연장됩니다.
</Warning>

<br />

뒷타임 회의의 참석자 목록이 조회됩니다. 연장 양해 메시지를 전송할 대상자를 선택한 뒤 `다음` 버튼을 클릭합니다.
<Frame>
    <img src="/images/tablet/meeting-extend/request-extend/img3.png" width="100%"/>
</Frame>

<br />

`연장 및 요청` 버튼을 선택합니다. 우선 <strong>연장 가능한 시간이 있다면 해당 시간만큼 우선 연장</strong>되며, <strong>선택한 뒷타임 참석자에게 연장 요청 슬랙 메시지가 전송</strong>됩니다.
<Frame>
    <img src="/images/tablet/meeting-extend/request-extend/img4.png" width="100%"/>
</Frame>

<br />

## 2. 회의 연장 요청이 수락된 경우

회의 연장 요청이 수락된 경우 태블릿에 연장 요청 수락 알림창이 띄워지며, <u>회의실 이용 시간이 연장됩니다.</u>
<Frame>
    <img src="/images/tablet/meeting-extend/request-extend/img5.png" width="100%"/>
</Frame>
<br />
<Info>
    변경된 회의 시간은 구글 캘린더 일정에도 바로 반영됩니다.
</Info>

<br />

## 3. 회의 연장 요청이 거절된 경우

회의 연장 요청을 거절된 경우 태블릿에 연장 요청 거절 알림창이 띄워지며, <u>회의실 이용 시간이 연장되지 않습니다.</u>
<Frame>
    <img src="/images/tablet/meeting-extend/request-extend/img6.png" width="100%"/>
</Frame>
