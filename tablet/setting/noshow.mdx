## 1. 노쇼 기능을 사용할 때 회의 시작하는 방법

노쇼 기능을 사용하는 경우, 회의 시작 시간이 되면 대기 화면으로 이동합니다.<br/>
회의를 진행하기 전에 좌측 하단에 있는 `회의 시작하기` 버튼을 눌러줍니다.
<Frame>
    <img src="/images/tablet/setting/noshow/img1.png" width="100%" />
</Frame>
<br />
<Warning>
    만약 [마일 관리자 페이지](https://mile.mintlify.app/essentials/setting/noshow#3)에서 설정한 노쇼 처리 시간(기본 10분) 내에 시작하기 버튼을 누르지 못한 경우, 노쇼 처리 되어 자동으로 회의가 종료됩니다.
</Warning>

<br />
회의 시작을 누르시면 회의 진행 화면인 `on-air 화면`으로 이동됩니다.
<Info>
    `on-air 화면`에 대해 더 알고싶으시다면, <a href="https://mile.mintlify.app/tablet/setting/noshow#3-on-air">on-air 화면 자세히 보기</a>을 확인해주세요.
</Info>
<br />
<Frame>
    <img src="/images/tablet/setting/noshow/img2.png" width="100%" />
</Frame>

<br />
## 2. 노쇼 기능을 사용하지 않을 때 회의 시작하는 방법

노쇼 기능을 사용하지 않는 경우, 회의 시작 시간이 되면 바로 `on-air 화면`으로 이동됩니다. <br/>
<Frame>
    <img src="/images/tablet/setting/noshow/img3.png" width="100%" />
</Frame>

<br />
<Info>
    노쇼 기능을 사용하지 않는 경우에는 특별한 액션 없이, 회의 시작 시간이 되면 바로 회의를 진행하시면 됩니다.
</Info>

<br />
## 3. on-air 화면 자세히 보기

on-air화면은 회의가 시작될 때 전환되는 화면이며, 회의가 진행중임을 나타냅니다. <br />
해당 화면에서 `다음 회의 예약하기`, `연장하기`, `예약 현황 조회하기`, `회의 종료하기` 등의 기능을 수행하실 수 있습니다.

타이머 보기 옵션에 따라 on-air는 2가지 종류의 화면으로 전환될 수 있습니다.

### 3-1. 타이머가 켜져있는 On-Air 화면

<Frame>
    <img src="/images/tablet/setting/noshow/img4.png" width="100%" />
</Frame>

-   **예약 현황**: 해당 회의실의 예약 현황을 볼 수 있는 버튼입니다.
-   **바로 예약**: 회의 중에 후속 회의를 바로 예약할 수 있습니다. <a href="https://mile.mintlify.app/tablet/book/next">더 자세히 보기</a>
-   **연장하기**: 회의 시간이 부족할 때, 회의실 예약 시간을 연장할 수 있습니다. <a href="https://mile.mintlify.app/tablet/meeting-extend/extend">더 자세히 보기</a>
-   **타이머 보기**: 회의실 참여자의 선호도에 따라 타이머 보기를 설정하여 타이머 사용 여부를 설정할 수 있습니다..
-   **회의 종료**: 회의가 일찍 끝난 경우, 다른 임직원들이 회의실을 이용할 수 있도록 회의를 종료 처리하는 버튼입니다. <a href="https://mile.mintlify.app/tablet/end">더 자세히 보기</a>

<br />
