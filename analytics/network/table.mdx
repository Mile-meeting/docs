<br />
`Mile Anaytics`는 Mile의 서비스를 통해 수집한 데이터를 기반으로 다음과 같은 기능을 제공합니다.

-   회의실 이용 현황 데이터
-   임직원 관계망 데이터
-   방문자 관리 데이터

이번 페이지에서는 임직원 관계망 데이터를 표형식으로 확인하는 방법을 알려드립니다.

## 1. 임직원 관계망 데이터 페이지 접속하기

로그인 후 메인 페이지 좌측 사이드 바의 Mile Analytics 탭을 클릭하고, 목록 탭에서 임직원 관계망 데이터 를 선택한 후 서브 목록인 관계망 테이블 목록을 선택하여 해당 페이지에 접속합니다.

<Frame>
    <img src="/images/analytics/network/table/img1.png" width="100%" height="300" />
</Frame>

<br />

### 2. 임직원 관계망 데이터 조회하기

임직원 관계망 데이터를 조회하기 위한 필터를 설정합니다.

설정할 수 있는 필터는 `기간` `임직원`이 있으며, 필터를 설정한 후에 `적용` 버튼을 누르면 하단에 표가 표시됩니다.

<Frame>
    <img src="/images/analytics/network/table/img2.png" width="100%" height="300" />
</Frame>

<br />
