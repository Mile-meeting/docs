## 1. 임직원 관계망 데이터 페이지 접속하기

로그인 후 메인 페이지에서 좌측 사이드 바의 두 번째 탭, `애널리틱스` 아이콘을 클릭하여 해당 페이지로 이동합니다.
`애널리틱스` 탭은 회의실 예약 데이터를 기반으로 예약 현황 데이터 리포트와 임직원 관계망 데이터를 제공합니다.

-   [예햑 현황 데이터 리포트 보러가기](https://mile.mintlify.app/analytics/book-report)

<br />

<Frame>
    <img src="/images/analytics/network/graph/img1.png" width="100%" height="300" />
</Frame>

<br />

좌측 목록 화면에서 `임직원 관계망 데이터` 탭을 클릭하시면, `관계망 테이블`, `관계망 그래프`탭이 펼처집니다.

<br />

<Frame>
    <img src="/images/analytics/network/graph/img2.png" width="100%" height="300" />
</Frame>

<br />

임직원 관계망 데이터 페이지는 설정한 특정 기간동안 함께 참여한 회의 시간을 기준으로, 임직원 간의 관계를 분석하여 제공하는 서비스입니다. `관계망 테이블`은 해당 관계망 데이터를 표 형식으로 표시하고, `관계망 그래프`는 네트워크 그래프 형식으로 데이터를 시각화하여 보여줍니다.

관계망 테이블의 사용 방법을 확인하시려면, [임직원 관계망 테이블 보기](https://mile.mintlify.app/analytics/network/table)를 클릭해주세요.

<br />

## 2. 관계망 그래프 필터 적용하기

관계망 테이블에서 설정할 수 있는 필터는 `기간`과, `임지원`이 있습니다.

<br />

<Frame>
    <img src="/images/analytics/network/graph/img3.png" width="100%" height="300" />
</Frame>

<br />

설정할 필터를 모두 설정하신 후에, `적용`버튼을 누르시면, 하단에 네트워크 그래프 형식으로 관계망 데이터가 보여집니다.

<Info>현재, 각 직원당 최대 10명의 상위 관계자와의 관계 그래프를 확인할 수 있습니다.</Info>

<br />

<Frame>
    <img src="/images/analytics/network/graph/img4.png" width="100%" height="300" />
</Frame>

<br />

`임직원 이름 탭`에서 관계망 그래프를 보고자하는 임직원을 클릭하면, 좌측에 해당 임직원의 관계망 그래프가 그려집니다.

<br />
