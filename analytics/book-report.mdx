## 1. 애널리틱스 탭 접속하기

로그인 후 메인 페이지에서 좌측 사이드 바의 두 번째 탭, `애널리틱스` 아이콘을 클릭하여 해당 페이지로 이동합니다.
`애널리틱스` 탭은 회의실 예약 데이터를 기반으로 예약 현황 데이터 리포트와 임직원 관계망 데이터를 제공합니다.

-   [임직원 관계망 데이터 보러가기](https://mile.mintlify.app/analytics/network/table)

## 2. 예약 현황 데이터 리포트 필터 설정하기

예약 현황 데이터 리포트는 회의실 예약 데이터를 기반으로, 원하는 필터에 맞게 데이터를 가공하여 표와 CSV 파일 형식으로 내려받을 수 있습니다.

데이터를 조회할 때 설정해야 하는 값은 보고 싶은 데이터의 `기간`, 원하는 컬럼에 해당하는 `항목`, 그리고 데이터에 조건을 적용할 수 있는 `필터`가 있습니다.

<Warning>필터는 선택한 항목에 대해서만 사용할 수 있습니다. 그러므로 필터를 설정하기 전에 원하는 항목을 먼저 선택해주세요.</Warning>

<br />
<Frame>
    <img src="/images/analytics/book-report/img2.png" width="100%" height="300" />
</Frame>

<br />

모든 설정을 완료한 후, `적용` 버튼을 누르시면 하단에 설정한 필터를 기준으로 표 형식으로 데이터를 확인하실 수 있습니다.

<Frame>
    <img src="/images/analytics/book-report/img3.png" width="100%" height="300" />
</Frame>

<br />

## 3. 에약 현황 데이터 CSV 다운 받기

엑셀에서 데이터를 활용하거나 추가 가공을 위해 CSV 형식으로 다운로드 기능을 제공하고 있습니다. `CSV 다운` 버튼을 클릭하시면 설정한 필터값에 해당하는 데이터를 다운로드 받을 수 있습니다.
