## 1. 애널리틱스 탭 접속하기

로그인 후 메인 페이지에서 좌측 사이드 바의 두 번째 탭, `애널리틱스` 아이콘을 클릭하여 해당 페이지로 이동합니다.
`애널리틱스` 탭은 회의실 예약 데이터를 기반으로 예약 현황 데이터 리포트와 임직원 관계망 데이터 방문객 데이터 를 제공합니다.

-   <a href="https://mile.mintlify.app/analytics/mile-meeting/table">임직원 관계망 데이터 보러가기</a>
-   <a href="https://mile.mintlify.app/analytics/mile-meeting/book-report">회의실 예약 데이터 보러가기</a>

`애널리틱스` 에서 `방문자 관리 데이터` 탭을 클릭하면, 방문자 관리 데이터를 조회해 볼 수 있는 페이지에 접속할 수 있습니다.

## 2. 방문자 데이터 조회하기

데이터를 조회할 기간을 필터로 설정한뒤에, 우측 `적용` 버튼을 누르면 방문자 데이터들을 조회할 수 있습니다.

해당 테이블에서 방문객의 **예약 경로, 방문 목적, 방문 시각, 부서 등의 다양한 메타 정보**를 확인할 수 있습니다.

<Frame>
    <img src="/images/analytics/visitor-report/img2.png" width="100%" />
</Frame>

<br />

## 3. 방문자 데이터 CSV 다운받기

엑셀에서 데이터를 활용하거나 추가 가공을 위해 CSV 형식으로 다운로드 기능을 제공하고 있습니다. `CSV 다운` 버튼을 클릭하시면 설정한 기간에 해당하는 데이터를 다운로드 받을 수 있습니다.
